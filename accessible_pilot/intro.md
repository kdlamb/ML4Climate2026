# ML4Climate 2026 — Accessible Pilot

*This is a screen-reader-friendly version of the course site for* **Machine
Learning for Climate Science and Environmental Sustainability** *(Columbia
Climate School, CLMT5403).* The [standard course
site](https://kdlamb.github.io/ML4Climate2026/) is the same material; this
version is restructured so a screen reader can read all of it.

## How this version is different

The standard site runs code in the **in-browser JupyterLab / Colab notebook
interface**, which is highly visual and hard to use with a screen reader. This
version keeps the same lessons and the same Python, but changes three things:

- **Workflow.** Instead of notebooks in a browser, you work in **VS Code with
  Python scripts** and a screen-reader-readable terminal. This is set up once
  and reused for the whole course.
- **Figures → text.** Every figure is replaced with a **text description** of
  what it shows, and formulas are written as readable math. Plots you generate
  yourself are explored with **MAIDR** (sonification + a text/braille readout),
  described in the setup guides below.
- **Notebooks → scripts.** The data-preprocessing tutorial and Assignment 1 are
  rewritten as linear scripts you run top to bottom, rather than notebook cells.

## Set up your workflow first (from CLMT5045)

The accessible workflow — installing VS Code, NVDA, a readable terminal, and the
MAIDR plotting tool — was built for the prerequisite course, **CLMT5045**, and
applies unchanged here. **Do these first** if you haven't already:

- **[Setting up an accessible
  workflow](https://earth-ds-ml.github.io/summer_2026/accessible/lectures_DS/computing_env/accessible_setup.html)**
  — VS Code + NVDA + a screen-reader-readable terminal, and connecting to the
  LEAP hub.
- **[Reading Python out
  loud](https://earth-ds-ml.github.io/summer_2026/accessible/lectures_DS/core_python/reading_python_out_loud.html)**
  — how code, indentation, and symbols sound with a screen reader.
- **[Accessible plots with
  MAIDR](https://earth-ds-ml.github.io/summer_2026/accessible/lectures_DS/sci_python/trying_maidr.html)**
  — hearing and reading a Matplotlib chart instead of seeing it.

If anything on those pages doesn't behave as described on your machine, that's
useful to tell the instructor or the course assistant.

## Course description

This course introduces the fundamental concepts and methods of machine learning
(ML), emphasizing practical applications to climate science and environmental
sustainability. You will gain both theoretical knowledge and practical coding
skills, working with real-world datasets to understand the capabilities and
limitations of ML in climate research. Topics include data preprocessing, model
selection, evaluation metrics, and the ethical implications of ML in climate
science.

## Course objectives

By the end of this course, you will:

- Develop a strong foundation in ML concepts and techniques applicable to
  climate and environmental challenges.
- Apply ML methods to analyze observational datasets, predict trends, and
  identify patterns.
- Critically assess the strengths and limitations of ML for climate and
  environmental applications.
- Understand ethical and policy implications of ML in climate research.
- Explore future trends and emerging technologies.

## Getting help and accommodations

This accessible version is a pilot. If you hit a page that isn't usable with
your screen reader, contact the instructor and Columbia's Disability Services so
we can fix it. Setup steps that need a one-time sighted hand (installing
software, the LEAP hub access token) can be done with the course assistant or
Disability Services.
